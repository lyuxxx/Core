Overview

Observer with block (KVO).

– addObserverBlockForKeyPath:block:

Registers a block to receive KVO notifications for the specified key-path relative to the receiver.

- (void)addObserverBlockForKeyPath:(NSString *)keyPath block:(void ( ^ ) ( id _Nonnull obj , id _Nonnull oldVal , id _Nonnull newVal ))block

Parameters

keyPath

The key path, relative to the receiver, of the property to observe. This value must not be nil.
block

The block to register for KVO notifications.

Discussion

The block and block captured objects are retained. Call removeObserverBlocksForKeyPath: or removeObserverBlocks to release.
